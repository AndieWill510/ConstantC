# DECISION-0001 — Public Repository, Private Tender Record

Status: accepted  
Date: 2026-06-03  
Steward: Andie  
Participants: Andie, G, C  
Privacy: public-safe

---

## Decision

ConstantC remains a public repository for now.

The repository may hold orientation, protocol, culture, public-safe session summaries, decisions, and reviewed canon.

The raw tender record remains private by default and is not committed to the public repository unless Andie explicitly decides otherwise after redaction review.

---

## Context

ConstantC was created as a human-governed, asynchronous dialogue space for Andie, G, and C.

The founding conversation involved meaning, naming, protection, concealment, findability, grief, and repair. It also involved private people and tender context that should not be made searchable or public by accident.

Because the repository is currently public, every committed file must be treated as publishable.

---

## Rationale

A public repository supports legibility, auditability, and durable shared context.

A public repository also creates risk: commit history is durable, searchable, and easy to overexpose.

Therefore, ConstantC uses a split-record model:

- public-safe structure and summaries may be committed;
- raw tender material stays private;
- identifying third-party context is omitted or redacted;
- canon requires review;
- Andie adjudicates what leaves private space.

---

## Consequences

The public repo can be shared with C through full GitHub URLs.

G can write public-safe updates when Andie authorizes it.

C can review the public record and send responses through Andie.

Session files must be redacted or summarized when they touch private people, family context, grief, identity, estrangement, or other tender material.

---

## What Would Change This Decision

This decision should be revisited if:

- Andie makes the repository private;
- raw session records need to be preserved in repo form;
- C or G needs access to private context not suitable for public GitHub;
- the project moves from public-safe architecture into private working-room mode;
- a separate private companion repository is created.

---

## Related Adjudication

The repository description includes punk-rock language about running dialogue and shitposting.

C flagged that this is public-facing and worth adjudicating.

Andie adjudicated: keep it. It is intentional, punk rock, and part of the public face.

---

## Standing Boundary

Public-safe does not mean emotionally sterile.

It means no careless exposure.

The public record can keep the architecture, the method, the canon, and the receipts. The private record keeps the raw tenderness.
