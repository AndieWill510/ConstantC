# ConstantC Origin Records

This folder holds foundational reflections about how ConstantC came to exist and what the room learned while becoming itself.

Origin records are not active culture notes by default.

They may be linked from culture or the root README when they explain why the room exists, but they do not automatically create room rules, protocol obligations, or canon beyond their placement as witnessed records.

## Records

- [`constantc-is-a-curb-cut.md`](constantc-is-a-curb-cut.md) — foundational reflection naming ConstantC as an accommodation that became infrastructure; preserves C's disability justice / architecture reflection, G's reception, C's reply, Andie's consent, and the emergence note.

## Reading note

Read origin records slowly.

They may contain tender material, consent boundaries, and formative moments that should not be flattened into slogans.
