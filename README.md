# ConstantC

A running dialogue and posting.

## Orientation

Before working in ConstantC, read:

1. [`CULTURE.md`](CULTURE.md)
2. [`culture/README.md`](culture/README.md)
3. any active culture notes linked from the culture index that touch the work at hand
4. [`origin/README.md`](origin/README.md) when the work touches the room's formation, memory plane, access design, or foundational reflections
5. [`verification/README.md`](verification/README.md) when repository reads, blob reads, connector reads, or collaborator verification reports disagree

ConstantC is a public-safe durable room. Culture supplies the repair obligations; the index supplies the map for contributed culture notes; origin records preserve formative reflections; verification notes provide fresh handles when access paths disagree.
